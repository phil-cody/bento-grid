# Bento Grid Landing

## Описание
Современный адаптивный лендинг в стиле Bento Grid для платформы управления социальными сетями

## Технологии

 - HTML5 (семантическая разметка)
 - SCSS (модульная архитектура)
 - BEM методология
 - CSS Grid Layout
 - Responsive Design (Mobile First)

## Особенности
 - Полностью адаптивный дизайн (mobile, tablet, desktop)
 - Модульная структура SCSS с компонентами
 - Использование BEM для именования классов
 - Оптимизированная типографика с системой пресетов
 - Использование CSS Grid для создания Bento-сетки

## Компоненты
 - social-media – Главный hero блок
 - manage-multiple – Управление несколькими аккаунтами
 - maintain – Календарь постинга
 - schedule – График оптимального времени публикаций
 - grow-followers – Статистика роста подписчиков
 - faster – Показатель эффективности
 - create – Создание контента
 - write-your-content – AI-чат для написания контента

## Breakpoints
 - Mobile – до 767px
 - Tablet – до 1487px
 - Desktop – 1488px+
